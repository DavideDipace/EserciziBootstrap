Risposta 1

A cosa serve bootstrap?

Bootstrap è un framework front-end gratuito per uno sviluppo web più rapido e semplice
Bootstrap include modelli di progettazione basati su HTML e CSS
per tipografia, moduli, pulsanti, tabelle, navigazione, modali, caroselli di immagini e molti altri, oltre a plugin JavaScript opzionali
Bootstrap ti dà anche la possibilità di creare facilmente progetti reattivi

Risposta 3

Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

Un CDN, o Content Delivery Network, è una rete di server distribuiti in diverse località geografiche progettata per consegnare contenuti digitali, come pagine web,
file CSS, JavaScript, immagini e video, in modo più veloce ed efficiente agli utenti finali.
Questa rete di server serve come intermediario tra il server web originale (dove è ospitato il tuo sito web) e gli utenti che accedono al sito. Alcuni vantaggi del Cdn sono : - Velocità di caricamento - Affidabilità - Risparmio di larghezza di banda - Caching - Sicurezza
